**Requirements:**
- TypeScript 5.4 or newer.
- Node.js, Deno, and Bun are supported.

<Aside type="note" title="Note">
  It is recommended to enable `strict: true` and `exactOptionalPropertyTypes:
  true` in your `tsconfig.json` file.
</Aside>

## Manual Installation

<Tabs syncKey="package-manager">
  <TabItem label="pnpm" icon="pnpm">
  ```sh showLineNumbers=false
  pnpm add redux-sonnet effect
  ```
  </TabItem>
  <TabItem label="npm" icon="seti:npm">
  ```sh showLineNumbers=false
  npm add redux-sonnet effect
  ```
  </TabItem>
  <TabItem label="Yarn" icon="seti:yarn">
  ```sh showLineNumbers=false
  yarn add redux-sonnet effect
  ```
  </TabItem>
</Tabs>
